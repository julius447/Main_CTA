Main CTA — 3 riktningar

Alla tre: nya Edvin-bilden, balanserad bild på mobil/iPad, centrerad mobilrubrik, vänsterställd brödtext, nya Google-badgen (g-row-mönstret). Skillnaden är CTA-hierarkin, proof-placeringen och gradienten.

Version A

Ring direkt

Löftet och handlingen pekar åt samma håll: "Ring 010-265 79 79" är primär (teal), "Kontakta oss" sekundär ghost. Gradient på "inom 60 sekunder!". Vågen kvar som brandmotiv.

 Version B

Proof först

Google-badgen som eyebrow ÖVER rubriken, EN primär CTA (Kontakta oss) + ring som textlänk, namnchip "Edvin · Elektriker, Ampy" på fotot. Gradient på värdelöftet.

 Version C

Utan gradient — A/B-kandidaten

Ingen gradient: rubriken bär hierarkin med vikt (400 + 700), teal används EN gång — på primärknappen. Inga vågor. Renaste ytan; testar om gradienten alls behövs.
Baslinjen (1:1-klonen av live) ligger kvar här.
